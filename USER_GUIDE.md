## Managing Clients

### Why Manage Clients?

Clients are the foundation of your CMS. Before you can create campaigns or content, you need to add your clients to the system.

### Viewing Your Clients

1. Click **"Clients"** in the sidebar
2. You'll see a table with all your clients
3. Each row shows:
   - Company logo (if uploaded)
   - Company name
   - Contact person
   - Email
   - Phone
   - Partner ID
   - Action buttons (Edit, Delete)

### Adding a New Client

**Step-by-Step:**

1. Click the **"+ Add Client"** button (top right, blue button)
2. A form will appear (called a "modal")
3. Fill in the information:

   ```
   Company Name: [Example Corp________]
   Contact Person: [John Doe__________]
   Email: [john@example.com___]
   Phone: [+1234567890________]
   Partner ID: [PART001___________]
   Logo: [Choose File] [No file chosen]
   ```

4. **Required fields:**
   - Company Name (must fill this)
   - All other fields are optional

5. **Uploading a Logo:**
   - Click **"Choose File"**
   - Select an image from your computer
   - Supported formats: JPG, PNG, GIF, WEBP
   - Maximum size: 5MB
   - You'll see a preview after selecting

6. Click **"Add Client"**

7. Success! You'll see a green message: âœ… "Client added successfully!"

### Editing a Client

**To edit existing client information:**

1. Find the client in the list
2. Click the **blue pencil icon** (Edit button)
3. A form appears with current information pre-filled
4. Make your changes
5. Click **"Save Changes"**

**Pro Tip:** You can change the logo by uploading a new one!

### Deleting a Client

âš ï¸ **Warning:** Deleting a client will also delete all their campaigns, content, and payments!

1. Find the client in the list
2. Click the **red trash icon** (Delete button)
3. A confirmation dialog appears
4. Click **"Yes, delete it!"** to confirm

### What is a Campaign?

A campaign is an advertising initiative you run for a client. It could be:
- Facebook ads
- Instagram promotions
- Google Ads
- LinkedIn campaigns
- Any platform-based marketing effort

### Viewing Campaigns

1. Click **"Campaigns"** in the sidebar
2. See all campaigns in a table format
3. Filter by client using the dropdown:
   ```
   Filter by Client: [All Clients â–¼]
   ```

### Adding a New Campaign

**Step-by-Step Guide:**

1. **Click "Add Campaign"** button

2. **Select the Client:**
   ```
   Select Client: [Choose Client â–¼]
   ```
   - Click the dropdown
   - Select the client this campaign is for

3. **Campaign Name:**
   ```
   Campaign Name: [Summer Sale 2025_____]
   ```
   - Enter a descriptive name

4. **Platform:**
   ```
   Platform: [Facebook â–¼]
   ```
   - Choose from: Facebook, Instagram, Google, LinkedIn, Twitter, TikTok, etc.

5. **Budget Information:**
   ```
   Budget: [5000______] (in Rs)
   Start Date: [2025-01-01]
   End Date: [2025-01-31]
   ```

6. **Performance Metrics:**
   ```
   Impressions: [50000____] (How many times the ad was shown)
   Reach: [25000____] (How many unique people saw it)
   Results: [1500____] (Clicks, conversions, etc.)
   Cost Per Result: [3.33____] (Rs)
   Total Spend: [5000____] (Rs)
   ```

   **What do these mean?**
   - **Impressions**: Total number of times your ad appeared
   - **Reach**: Number of unique users who saw your ad
   - **Results**: Actions taken (clicks, likes, conversions)
   - **Cost Per Result**: How much each action cost
   - **Total Spend**: Total money spent on the campaign

7. **Creative Images:**
   ```
   Creative Images: [Choose Files]
   ```
   - Click **"Choose Files"**
   - Select multiple images (the actual ads/creatives)
   - You can upload more than one!
   - Preview appears after selection

8. **Evidence Images:**
   ```
   Evidence Images: [Choose Files]
   ```
   - Upload screenshots of performance data
   - Upload analytics screenshots
   - Upload any proof of results
   - Label each image (e.g., "Ad Performance", "Analytics Data")

9. **Description:**
   ```
   Description:
   [Campaign details go here...
   Target audience, goals, notes...]
   ```

10. **Click "Add Campaign"**

### Campaign Image Types Explained

**Creative Images** = The actual ads you ran
- Ad designs

**Evidence Images** = Proof of performance

## Frequently Asked Questions

### General Questions

**Q: Is my data safe?**
A: Yes! The system uses:
- Encrypted passwords
- Secure connections
- Regular backups (by admin)
- User access controls

**Q: Can I access this from my phone?**
A: Yes! The system is mobile-responsive. However, for the best experience, we recommend using a laptop or desktop computer.

**Q: Can multiple people use the system at once?**
A: Yes! Multiple users can be logged in and working simultaneously.

**Q: What happens if I lose internet connection?**
A: You can still view data you've already loaded (it's cached locally), but you won't be able to save new data or load new information until your connection is restored.

**Q: How do I get my own account?**
A: Contact your system administrator. They can create an account for you.

### Client Management

**Q: Can I have multiple contacts for one client?**
A: Currently, you can list one primary contact person. For additional contacts, add them to the notes field or contact your admin about custom requirements.

**Q: What if I don't have a client's logo?**
A: That's fine! The logo is optional. You can add it later when you get it.

**Q: Can I merge duplicate clients?**
A: Not automatically. Contact your admin to manually merge client records.

### Campaign Management

**Q: How many images can I upload per campaign?**
A: You can upload multiple images for both creative and evidence. There's no hard limit, but we recommend keeping it reasonable for performance (10-20 images per campaign).

**Q: What if I don't know all the metrics yet?**
A: You can save a campaign with partial data and edit it later to add metrics as they become available.

**Q: Can I copy a campaign?**
A: Not currently. You'll need to create a new campaign and manually enter the details.

### Content & Finance

**Q: What's the difference between Content and Campaigns?**
A: 
- **Content** = Creative pieces you produce (posts, graphics, videos)
- **Campaigns** = Paid advertising campaigns (Facebook Ads, Google Ads, etc.)

**Q: Who can add payments?**
A: Only Admin users can add, edit, or delete payment records for security reasons.

**Q: Can I track payments in different currencies?**
A: Currently, the system uses a single currency (Rs). Contact your admin about multi-currency support.

### Reports & Invoices

**Q: Can I customize the invoice template?**
A: The basic layout is fixed, but you can customize:
- Company information
- Logo
- Payment terms
- Notes section

**Q: Can I email reports directly from the system?**
A: Not currently. Generate the PDF and email it using your email client.

**Q: How long are invoices kept in the system?**
A: Invoices are stored indefinitely unless manually deleted by an admin.
